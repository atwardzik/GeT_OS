NAME

bind - assign a port to a socket

SYNOPSIS

#include <socket.h>

int bind(int sockfd, const struct sockaddr *addr, size_t addrlen);

DESCRIPTION

bind() assigns port to a socket.

PARAMETERS

  • sockfd - descriptor of a socket
  • addr - source socket address
  • addrlen - size of addr

RETURN VALUES

Upon successful completion, a value of 0 is returned. Otherwise, negative error code is returned.

ERRORS

  • [EBADF] - sockfd is not a valid file descriptor

CODE EXAMPLES

struct sockaddr_in source = {.sin_family = AF_INET,
                             .sin_addr.s_addr = 0,
                             .sin_port = htons(8080)};

if (bind(sockfd, (struct sockaddr *) &source, sizeof(source)) < 0) {
        dprintf(2, "Error while trying to bind\n");
}
