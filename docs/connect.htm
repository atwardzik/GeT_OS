NAME

connect - initiate a connection o a socket

SYNOPSIS

#include <socket.h>

int connect(int sockfd, struct sockaddr *addr, size_t addrlen);

DESCRIPTION

connect() initiates a connection on a bound socket.

PARAMETERS

  • sockfd - descriptor of a socket
  • addr - destination socket address
  • addrlen - size of addr

RETURN VALUES

A negative error code is returned if an error occurs, otherwise 0.

ERRORS

  • [EBADF] - sockfd is not a valid file descriptor
  • [ETIMEDOUT] - Connection timeout occured
  • [ECONNABORTED] - There are no free hardware sockets

CODE EXAMPLES

struct sockaddr_in dest = {AF_INET, htons(8080)};
inet_aton("192.168.1.40", &dest.sin_addr);

if (connect(sockfd, (struct sockaddr *) &dest, sizeof(dest)) < 0) {
        dprintf(2, "Error while trying to connect - connection timed out\n");
        close(sockfd);
}
