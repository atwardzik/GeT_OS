NAME

socket - create an endpoint for communication

SYNOPSIS

#include <socket.h>

int socket(int domain, int type, int protocol);

DESCRIPTION

socket() creates an endpoint for communication and returns a descriptor.

PARAMETERS

  • domain - currently only AF_INET corresponding to ipv4 protocol
  • type - one of the following communication types:
    • SOCK_STREAM - for TCP communication
    • SOCK_RAW - for RAW communication
  • protocol - ignored

RETURN VALUES

A negative error code is returned if an error occurs, otherwise the return value is a descriptor referencing the socket.

ERRORS

  • [EAFNOSUPPORT] - The specified address family is not supported
  • [ENFILE] - The system file table is full
  • [EPROTONOSUPPORT] - The protocol is not supported
  • [ENOSOCKFREE] - There are no free hardware sockets

CODE EXAMPLES

int sockfd = socket(AF_INET, SOCK_STREAM, 0);
if (sockfd < 0) {
        dprintf(2, "Error while trying to open socket\n");
}
For full socket-based connections establishment please refer to the internet.
