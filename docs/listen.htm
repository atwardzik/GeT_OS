NAME

listen - listen for connections on a socket

SYNOPSIS

#include <socket.h>

int listen(int sockfd, int backlog);

DESCRIPTION

listen() - express a willingness to listen for incoming connection after successful TCP (SOCK_STREAM) socket creation with socket() and port assignment with bind().

PARAMETERS

  • sockfd - descriptor of a socket
  • backlog - ignored

RETURN VALUES

Upon successful completion, a value of 0 is returned. Otherwise, negative error code is returned.

ERRORS

  • [EBADF] - sockfd is not a valid file descriptor

CODE EXAMPLES

struct sockaddr_in source = { ... };

if (bind(sockfd, (struct sockaddr *) &source, sizeof(source)) < 0) { ... }

if (listen(sockfd, 0) < 0) {
        dprintf(2, "Error while trying to listen on port: %i\n",
                ntohs(source.sin_port));
}
