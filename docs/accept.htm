NAME

accept - accept an incoming connection

SYNOPSIS

#include <socket.h>

int accept(int sockfd, struct sockaddr *addr, size_t addrlen);

DESCRIPTION

accept() - accepts an incoming connection after successful socket creation with socket() and port assignment with bind().

PARAMETERS

  • sockfd - descriptor of a socket
  • addr - destination socket address
  • addrlen - size of addr

RETURN VALUES

Upon successful completion, a socket number is returned for POSIX compliancy, however, the returned socket is the same as the socket that listens for the connection - the hardware socket establishes a connection and is blocked until the connection is closed. Otherwise, negative error code is returned.

ERRORS

  • [EBADF] - sockfd is not a valid file descriptor

CODE EXAMPLES

struct sockaddr destination = {};

int destfd;
if ((destfd = accept(sockfd, &destination, sizeof(destination))) < 0) {
        dprintf(2, "Error while trying to accept connection\n");
}
